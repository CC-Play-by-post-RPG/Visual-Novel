This repository will hold the code for the Visual Novel based on the play-by-post RPG found on the CC forums.

#How to clone and use this repo
* Go to http://www.renpy.org/latest.html and download the SDK.
* Install it into any folder you like.
* Open renpy and create a new project.
* Once you are done locate the root directory of the new project. Then go into the "game" folder.
* Delete everything in that folder. Then clone this repo into the now empty folder.
